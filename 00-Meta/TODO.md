# Temp
### git
- reset -> --mix는 staging area에 있는 스냅샷 blob을 삭제(정확히는 index에 있는 포인터를 삭제. 실제 blob은 GC가 삭제한다고 함), 현재 파일 시스템 내에 있는 working Directory의 파일은 그대로 유지
- reset -> --soft는 staging 그대로 유지(커밋할 내용이 있는 공간임). 파일도 그대로 유지 -> 커밋이 분화되는거임. 
- reset -> --hard는 staging을 이전 커밋 (이전 blob으로 복구), 파일도 이에 따라 이전 버전으로 복구됨(영구적)
