# 사례-IntelliJ 프로젝트 탐색기 줄간격 수정 방법

## 1. 개요
IntelliJ IDEA의 프로젝트 탐색기(Project Explorer)에서 파일 간의 줄 간격(Row Height)이 너무 좁거나 넓을 때, 이를 사용자 지정 속성을 통해 원하는 수치로 조정하는 방법입니다.

---

## 2. 해결 방법

### 2.1. 사용자 지정 속성 편집기 실행
설정 파일을 열기 위해 다음 두 가지 방법 중 하나를 선택합니다.

- **메뉴 경로**: `도움말(Help)` > `사용자 지정 속성 편집...(Edit Custom Properties...)`
- **단축키 활용 (추천)**: 
    1. `Shift` 키를 연속으로 두 번 눌러 전체 검색(Search Everywhere) 실행
    2. `Edit Custom Properties` 입력 후 해당 항목 선택

### 2.2. 속성 값 추가 및 적용
파일이 열리면 가장 아랫줄에 다음 내용을 입력하고 저장합니다.

```properties
# 프로젝트 트리 행 높이 설정 (기본값은 폰트 크기에 따라 자동 결정됨)
ide.override.Tree.rowHeight=22
```

> [!TIP]
> 22~24 사이의 값이 가장 보편적으로 사용되며, 사용자의 모니터 해상도와 취향에 따라 조절하십시오.

---

## 3. 결과 및 확인
1. 설정을 저장한 후 IntelliJ를 **재시작**합니다.
2. 프로젝트 탐색기의 줄 간격이 설정한 수치(`22`)로 고정되어 가독성이 개선된 것을 확인합니다.
