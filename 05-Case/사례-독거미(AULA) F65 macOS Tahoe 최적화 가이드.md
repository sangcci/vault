# 사례-독거미(AULA) F65 macOS Tahoe 최적화 가이드

## 1. 개요
독거미(AULA) F65 키보드를 macOS Tahoe 환경에서 최적화하여 사용하기 위한 설정 가이드입니다. 특히 65% 배열의 특성상 부족한 키 매핑과 입력 최적화 설정을 중점적으로 다룹니다.

## 2. 해결 과정

### 2.1. 펌웨어 및 시스템 모드 설정
펌웨어 업데이트 도구는 **Windows 전용**이므로, 반드시 PC에서 업데이트를 먼저 완료해야 합니다.

- **Mac 모드로 전환**: `Fn + D`
    - 하단 배열 키 재매핑: Alt → Option, Win → Command
- **Windows 모드로 전환**: `Fn + W`

### 2.2. 키보드 조명 설정 (화이트 LED)
전용 소프트웨어 없이 하드웨어 단축키로 설정합니다.

- **색상 변경**: `Fn + Tab` (흰색이 될 때까지 반복)
- **패턴 변경**: `Fn + ` (단색 고정 모드 권장)
- **밝기 조절**: `Fn + ↑` / `Fn + ↓`

### 2.3. 코딩 필수 키 입력 (Backtick & Tilde)
65% 배열에서 ESC 키와 공유되는 기능을 입력하는 방법입니다.

- **백틱 ( ` )**: `Fn + Esc`
- **물결 ( ~ )**: `Fn + Shift + Esc`
- **한글 모드 팁**: `Option + Fn + Esc`를 누르면 한글 입력 상태에서도 백틱(`)을 강제 입력할 수 있습니다.
