# 개념: 가상 네트워크 (Virtual Network)

## 📝 정의
물리적인 네트워크 장비 위에 소프트웨어 설정을 통해 논리적으로 분리된 네트워크 망을 구축하는 기술입니다. 클라우드의 **VPC(Virtual Private Cloud)**가 대표적입니다.

## ⚙️ 핵심 원리

### 1. 오버레이 네트워크 (Overlay Network)
- 물리 네트워크 위에 터널링 기술을 사용하여 가상의 연결을 만듭니다.

### 2. 논리적 격리
- 동일한 물리 장비에 연결되어 있어도, 가상 네트워크가 다르면 서로 통신할 수 없도록 차단하여 보안을 강화합니다.

## 💡 한 줄 요약
**가상 네트워크**는 커다란 운동장(물리망)에 보이지 않는 **'투명한 칸막이'**를 쳐서, 서로 다른 팀들이 같은 공간에 있으면서도 섞이지 않게 만드는 기술이다.
