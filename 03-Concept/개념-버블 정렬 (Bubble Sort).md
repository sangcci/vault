## 한 문장 정의

> (사전적) 인접한 두 원소를 비교해 교환하며, 큰 값이 끝으로 밀려나도록 반복하는 정렬 방식.
> (이해용) 물속 거품처럼 큰 값이 매 단계마다 오른쪽 끝으로 떠오르는 것.

---

## 해결하는 문제

- 구현이 가장 단순한 정렬이 필요한 경우

---

## 치르는 비용

- 평균/최악 시간복잡도: O(n²)
- 최선 시간복잡도: O(n) — 교환이 한 번도 없으면 조기 종료 가능
- 공간복잡도: O(1) — 제자리 정렬
- 교환 횟수가 삽입/선택 정렬보다 많아 실제로는 가장 느린 편

---

## 동작 원리

**핵심**: 매 단계마다 인접 원소를 비교해 큰 값을 오른쪽으로 밀어냄. 한 단계가 끝나면 그 단계의 최댓값이 맨 오른쪽에 확정.

```
초기: [5, 3, 4, 1, 2]

step 1: 인접 비교 및 교환
        5>3 → [3,5,4,1,2]
        5>4 → [3,4,5,1,2]
        5>1 → [3,4,1,5,2]
        5>2 → [3,4,1,2,5]  ← 5 확정

step 2: [3,4,1,2] 반복
        4>1 → [3,1,4,2,5]
        4>2 → [3,1,2,4,5]  ← 4 확정

step 3: [3,1,2] 반복
        3>1 → [1,3,2,4,5]
        3>2 → [1,2,3,4,5]  ← 3 확정

오른쪽부터 거품처럼 최댓값이 쌓임
```

```python
for i in range(len(arr)):
    swapped = False
    for j in range(len(arr) - i - 1):
        if arr[j] > arr[j + 1]:
            arr[j], arr[j + 1] = arr[j + 1], arr[j]
            swapped = True
    if not swapped:   # 교환 없으면 조기 종료
        break
```
