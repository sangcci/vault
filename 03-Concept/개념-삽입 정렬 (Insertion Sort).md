## 한 문장 정의

> (사전적) 이미 정렬된 부분에 새로운 원소를 올바른 위치에 삽입하며 정렬하는 방식.
> (이해용) 손에 들고 있는 카드 패에 새 카드를 뽑을 때마다 제자리에 끼워 넣는 것.

## 해결하는 문제

- 소규모 데이터 또는 거의 정렬된 데이터를 효율적으로 정렬

## 치르는 비용

- 평균/최악 시간복잡도: O(n²) — 역순 정렬 시 모든 원소를 매번 비교
- 최선 시간복잡도: O(n) — 이미 정렬된 경우 각 원소를 한 번씩만 비교
- 공간복잡도: O(1) — 제자리 정렬

## 동작 원리

**핵심**: 인덱스 0은 이미 정렬된 상태로 간주. 인덱스 1부터 하나씩 꺼내 왼쪽 정렬 구간에서 제자리를 찾아 삽입.

```
초기: [5, 3, 4, 1, 2]

step 1: [5] | 3 꺼냄 → 5보다 작으므로 앞에 삽입
        [3, 5] | 4, 1, 2

step 2: [3, 5] | 4 꺼냄 → 5보다 작고 3보다 크므로 사이에 삽입
        [3, 4, 5] | 1, 2

step 3: [3, 4, 5] | 1 꺼냄 → 전부 앞으로
        [1, 3, 4, 5] | 2

step 4: [1, 3, 4, 5] | 2 꺼냄 → 1 뒤, 3 앞에 삽입
        [1, 2, 3, 4, 5]

왼쪽: 항상 정렬된 상태 유지
오른쪽: 아직 처리 안 된 원소들
```

```python
for i in range(1, len(arr)):
    key = arr[i]
    j = i - 1
    while j >= 0 and arr[j] > key:
        arr[j + 1] = arr[j]  # 한 칸씩 오른쪽으로 밀기
        j -= 1
    arr[j + 1] = key          # 빈 자리에 삽입
```
