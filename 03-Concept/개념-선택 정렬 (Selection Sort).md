## 한 문장 정의

> (사전적) 정렬되지 않은 구간에서 가장 작은 원소를 선택해 맨 앞으로 가져오며 정렬하는 방식.
> (이해용) 줄 세우기에서 매번 가장 작은 사람을 골라 앞에 세우는 것.

---

## 해결하는 문제

- 구현이 단순한 정렬이 필요한 경우

---

## 치르는 비용

- 평균/최선/최악 시간복잡도: O(n²) — 항상 남은 구간 전체를 탐색
- 공간복잡도: O(1) — 제자리 정렬
- 삽입 정렬과 달리 이미 정렬된 데이터에서도 O(n²)

---

## 동작 원리

**핵심**: 매 단계마다 미정렬 구간에서 최솟값을 선택해 미정렬 구간의 첫 번째 자리와 교환.

```
초기: [5, 3, 4, 1, 2]

step 1: 미정렬 [5,3,4,1,2] → 최솟값 1 선택 → 5와 교환
        [1] | [3, 4, 5, 2]

step 2: 미정렬 [3,4,5,2] → 최솟값 2 선택 → 3과 교환
        [1, 2] | [4, 5, 3]

step 3: 미정렬 [4,5,3] → 최솟값 3 선택 → 4와 교환
        [1, 2, 3] | [5, 4]

step 4: 미정렬 [5,4] → 최솟값 4 선택 → 5와 교환
        [1, 2, 3, 4, 5]

왼쪽: 확정된 최솟값들
오른쪽: 아직 선택 안 된 원소들
```

```python
for i in range(len(arr)):
    min_idx = i
    for j in range(i + 1, len(arr)):
        if arr[j] < arr[min_idx]:
            min_idx = j          # 최솟값 선택
    arr[i], arr[min_idx] = arr[min_idx], arr[i]  # 교환
```
