## 판단 기준 (Criteria)

> 초보 운전자가 출발 전·주행 중·후진 시 각각 어떤 기준으로 행동해야 하는가?

---

## 1. 시트 포지션 — "모든 관절이 구부러져 있는가?"

```
[올바른 자세]

  머리 — 헤드레스트에 닿음
   |
  어깨 — 시트 등받이에 밀착
   |
  팔꿈치 — 살짝 구부러진 상태 (핸들 12시 잡았을 때)
   |
  엉덩이 — 시트 깊숙이
   |
  무릎 — 브레이크 풀 밟아도 펴지지 않음  ← 핵심
   |
  발꿈치 — 바닥 고정, 발바닥 살로 페달 컨트롤
```

- 오른발 일직선 → **브레이크**에 정렬. 엑셀은 45도 틀어서 밟음
- 위급 상황 시 몸이 자동으로 밟는 방향 = 브레이크여야 함
- 시트 높이: 전방 시야 확보될 만큼 올림
- 시트 전후: 사이드미러가 자연스럽게 보일 만큼 조절

### 실패하는 상황
- 무릎이 완전히 펴지는 포지션 → 충돌 시 부상 위험 + 페달 컨트롤 불가
- 발꿈치가 너무 멀면 발바닥 살 컨트롤 불가 → 페달 on/off식 조작이 됨

---

## 2. 페달 조작 — "눌렀다 뗐다 하지 않는가?"

```
[엑셀 압력 vs 속도 관계]

압력 ─────►  유지        살짝 추가      유지
속도   ──────────►  ────────────►  ──────►
             선형 증가     기어 변속      일정 속도

[브레이크 감각]

풀 브레이크 ██████████  ← 절대 금지 (급제동)
지긋이      ████░░░░░░  ← 발바닥 살로 압력 조절
정지 직전   ██████████  ← 꾸욱 눌러 압 유지 (차 밀림 방지)
```

| 의도 | 행동 |
|:---|:---|
| 속도 높이기 | 엑셀 압력 살짝 추가 → 유지 |
| 속도 낮추기 | 엑셀 압력 살짝 줄이기 → 유지 |
| 서서히 멈추기 | 엑셀에서 발 떼기 → 브레이크로 옮기기 → 지긋이 밟기 |

### 핵심 원칙
- 엑셀: 압력 **유지**가 기본. 밟았다 뗐다 = 승객 멀미
- 브레이크: **발바닥 살**로 압력 조절. 무게중심 쏠림 방지
- RPM 게이지 항상 확인

---

## 3. 탄력 주행 — "엑셀에서 바로 브레이크로 가지 않는가?"

```
[정지 시퀀스]

  신호등 보임 / 마름모 표시
        │
        ▼
  ① 엑셀에서 발 뗌
        │
        ▼
  ② 브레이크로 발 옮김 (탄력으로 주행 중)
        │
        ▼
  ③ 브레이크 지긋이 → 정지
```

- 차는 엑셀을 떼도 탄력으로 계속 전진함
- 엑셀 → 즉시 브레이크 = 급제동 느낌. 탄력 구간을 활용해야 자연스러움

---

## 4. 차선 유지 — "기준점을 잡고 있는가?"

```
[방법 1: 중앙선 기준]

    ┌─────────────────┐
    │     차선 중앙     │
    │    (임의 선)      │
    │        ↕          │
    │  내 오른쪽 무릎   │
    └─────────────────┘

[방법 2: 왼쪽 차선 기준]

    왼쪽 차선 ──── 본네트 왼쪽 끝과 겹침
```

- 방법 1이 범용적. 차종 무관하게 적용 가능
- 방법 2는 차종마다 기준이 달라질 수 있음
- 양옆에 주차 차량이 있는 좁은 도로에서는 결국 **차폭감** 자체가 필요

---

## 5. 차폭감 — "직접 내려서 확인했는가?"

- 숏컷 없음. 반사 테이프 등 보조 수단이 있지만 결국 **반복 연습**
- 감이 안 잡힐 때: 차에서 내려 실제 차폭과 감각 차이를 직접 확인

---

## 6. 후진 — "뒷바퀴를 축으로 인식하고 있는가?"

```
[후진 회전 원리]

         ┌──── 앞바퀴 (크게 이동)
         │
    ┌────┴────┐
    │   차체   │
    └────┬────┘
         │
         └──── 뒷바퀴 (축 · 거의 안 움직임)

→ 뒷바퀴 기준점이 맞춰진 상태에서 핸들 틀기
→ 박을 것 같으면? 살짝 앞으로 빼서 각도 재조정
```

---

## 7. 90도 회전 (시내 주행)

- 일일 노트에 추가 내용 확인 후 보완 예정

---

## 효과적인 상황 (Effective When)

- 운전면허 취득 직후 도로 주행 시작 단계
- 학원 실습 중 기본기 점검용
- 장기간 운전 안 하다가 다시 시작할 때 복기용

## 실패하는 상황 (Fails When)

- 고속도로, 산길, 빗길 등 특수 환경에서는 추가 판단 기준 필요
- 수동 변속기(MT) 차량에는 페달 조작 기준이 다름
