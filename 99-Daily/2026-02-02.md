# 회고

토큰 값을 최대한 덜 사용하는 방법을 찾고 싶다. Local LLM을 활용해서 쉬운 작업은 Local이, 복잡한 작업만 Claude가 담당하도록 하는 구조가 있는데, 이를 한 번 찾아보자.

Claude Code 구독 제도에는 시간 제한도 존재한다. 일일 제한이 아닌 rolling window 방식으로 동작한다. 자세히 알아봐야 할 듯 하다.

Anki 연동을 마쳤고, Principle한 용어들부터 확실하게 10개씩 익혀나갈 계획이다. 이를 마치면 개념들을 차례로 익혀나갈 계획. 계속 시간 날때마다 볼 것이다.

AI가 날로 갈수록 발전해나가면서, 관련된 기술들이 정말 많이 불어났다. 세상의 흐름에 올라타기 위해선, 매일 AI에 대한 활용 방법도 1시간 정도 투자해서 학습해보자.
