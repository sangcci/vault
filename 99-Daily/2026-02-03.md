# 회고

- Claude Code에서 Gemini CLI로 환경을 옮겼는데, 결과물이 잘 나오는 것 같다. Claude Code와 다르게 limit 계산을 하루 단위로 초기화 하기 때문에 작업 시간을 계산하기 편하다. 단점은 결과물이 나올 때 까지 시간이 너무 오래 걸린다는점. 다른 작업을 하면서 병행해야 될 듯하다. 앞으로 Gemini CLI는 문서 정리 및 Anki 용도로 사용한다.
- 추가로 Gemini CLI 무료는 조금만 사용해도 금방 토큰 limit에 도달하는 것 같다...사야되나..?
- 결국 나 자신의 학습 속도에 AI가 맞춰져야 되는 것 같다. 이때까지 나는 개념 및 원리에 대한 이해와 나만의 언어로 정리하는 것을 너무 가볍게 여겼다. 때문에 문서들의 퀄리티는 괜찮았으나, Anki Card를 보면 낯설었다. 양 보다는 질. 결국 Card를 여러개를 동시에 생성하는 것 보다, 일일이 하나씩 직접 내가 검수하면서 추가해야 할 듯 하다.
- Principle은 내가 직접 용어 설명을 할 수 있어야 하지만, 가장 중요한 점은 "이것을 적재적소에 사용할 수 있어야 한다"는 점이다. 개발자들은 업계 용어들을 수도없이 사용하며 소통한다. 서로 다르게 이해해서 소통에 차질이 생기는 것을 방지하기 위해 업계 공통의 언어를 사용할 수 있어야 한다.
