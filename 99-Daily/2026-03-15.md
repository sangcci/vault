### 4. Docker 컨테이너 → virbr0 (KVM) 네트워크 차단 해결 (3/10)
**유형**: `사례 (05-Case)` → `[[사례-Docker에서 KVM virbr0 대역 접근 차단 해결]]`

- 원인: libvirt iptables 규칙이 docker0 → virbr0 FORWARD를 차단
- 해결: `iptables -I FORWARD` 양방향 ACCEPT + `netfilter-persistent save`
- ufw는 FORWARD 체인 직접 제어 불가

### 5. KVM VM 아웃바운드 차단 — MASQUERADE 규칙 누락 (3/10)
**유형**: `사례 (05-Case)` → `[[사례-KVM VM 외부 네트워크 차단 MASQUERADE 누락]]`

- 증상: VM에서 `curl https://github.com` hang, self-hosted runner offline
- 원인: libvirt가 자동 생성해야 할 NAT MASQUERADE 규칙 누락 (Docker 설치 후 iptables 덮어써짐)
- 해결: `iptables -t nat -A POSTROUTING -s 192.168.122.0/24 ! -o virbr0 -j MASQUERADE`

### 6. 그리디 알고리즘 개념 정리 (3/11)
**유형**: `개념 (03-Concept)` → `[[개념-그리디 알고리즘 (Greedy)]]`

- 각 단계에서 국소 최적 선택 → 전체 최적으로 이어지는 경우에 사용
- 핵심: "왜 이 기준이 최적인가"를 납득할 수 있어야 함 (교환 논증)
- 회의실 배정 (BOJ 1931): 끝시간 오름차순 정렬 → O(n log n)
- DP와 차이: DP는 부분 문제 저장 + 재활용, 그리디는 뒤돌아보지 않음
