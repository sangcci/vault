# 기록

### Flyway
적용 완료
- 파일 이름 규칙 어떻게 설정할건지 정해야 함
- flyway 실행 과정을 익힘. spring applicationContext가 초기화될 때(JPA EntityManagerFactory 초기화 직전에 수행) -> Flyway Bean 생성하고 여기서 flyway_schema_history 테이블을 조회함. -> PENDING 상태 SQL 파일을 순서대로 실행함. -> 완료 후 일반 앱 로직 이어서 시작.
- SQL파일을 직접 넣어야 된다는 사실을 알게 됨
- 추적하는 DDL
	- CREATE TABLE
	- CREATE SEQUENCE
	- ALTER TABLE ... ADD CONSTRAINT
	- CREATE INDEX
	- CREATE TYPE / CREATE ENUM
