### 정처기 정리
- java는 인스턴스가 static 맴버 변수 접근을 허용함. 권장된 방법은 아니라고 IDE가 경고 메세지를 띄울 뿐.
- %연산을 활용하는 문제 나옴. 앞자리 숫자를 꺼내는 방법은 `n % 10`임.
- SQL문은 attribute를 표현할 때 `'이름'`이 아닌 따옴표 없는 `이름`으로 표시.
- python에서 set 자료구조의 `.update()`는 기존에 존재하는 데이터는 놔두고 새로운것만 뒤에 추가.
- `a = b = c`에서 누가 먼저 실행됨??
- new Child()했을 때 생성자 내에 `super()`가 내장되어 생략되있음. 그래서 자식 클래스 생성자 호출 시 super()에 의해 부모 클래스도 같이 초기화되어 할당됨. 다만 접근은 Child를 통해서 할 수 있다는 점.
