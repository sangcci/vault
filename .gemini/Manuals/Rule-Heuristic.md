# Heuristic Note Rules (판단기준)

## Purpose
To document **decision-making strategies** or rules of thumb for engineers.

## File Naming
- Format: `06-Heuristic/판단기준-<Title>.md`
- Example: `판단기준-시스템 설계 체크리스트.md`

## Template
```markdown
---
aliases: []
tags: [판단기준, 작성중]
type: Heuristic
difficulty: High
---
# Heuristic: <Title>

## 판단 기준 (Criteria)
- When to choose A vs B?

## 효과적인 상황 (Effective When)
- ...

## 실패하는 상황 (Fails When)
- ...
```
