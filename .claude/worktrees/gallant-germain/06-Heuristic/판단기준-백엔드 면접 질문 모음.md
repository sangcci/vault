# Heuristic: 백엔드 면접 질문 모음

> **전략적 나침반**: 신입 백엔드 개발자 기술 면접에서 무엇이 중요한지, 어떤 질문이 나올지 파악하기 위한 가이드라인.

## Criteria (우선순위 가이드라인)
1. **신입 필수 (Must-Have)**: Java 기본, 자료구조, 알고리즘, DB 기초, 네트워크 기초, OS 기초.
2. **실무 역량 (Should-Have)**: Spring/Spring Boot, JPA, 동시성 제어, 보안 기초.
3. **심화/관심도 (Nice-to-Have)**: MSA, 인프라(Docker/K8s), 고급 GC 튜닝, 검색 엔진.

## Effective When
- 면접을 앞두고 전체적인 지식 지도를 점검할 때.
- 학습 로드맵을 설정하여 공부의 우선순위를 정할 때.

## Fails When
- 개별 기술의 깊이 있는 원리 파악 없이 질문-답변만 암기할 때. (Question/Concept 단계 병행 필수)
